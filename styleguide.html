Toggle button:

Noter og bemærkninger omkring brugen af elementet.

<label class="switch"> <input type="checkbox"> <span class="toggle"></span> </label>
.switch { position: relative; display: inline-block; width: 40px; height: 20px; }

.switch input {display:none;}

.toggle { position: absolute; cursor: pointer; top: 0; left: 0; right: 0; bottom: 0; background-color: #ccc; -webkit-transition: .3s; transition: .3s; border-radius: 34px; }

.toggle:before { position: absolute; content: ""; height: 16px; width: 16px; left: 3px; bottom: 2px; background-color: #fff; -webkit-transition: .3s; transition: .3s; border-radius: 50%; }

.switch input:checked + .toggle { background-color: #25a18e; }

.switch input:checked + .toggle:before { -webkit-transform: translateX(18px); -ms-transform: translateX(18px); transform: translateX(18px); }

.checkbox input[type=checkbox]:checked + label:after { opacity: 1; }
